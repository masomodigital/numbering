Learning HTML: Numbering and Bullet Points

Introduction

Welcome to the lesson on creating lists in HTML! You can use lists to organize items or information in a clear way.

Explanation

There are two main types of lists in HTML:

  1. Ordered Lists - These are numbered lists created with the <ol> tag.
  2. Unordered Lists - These are bullet points created with the <ul> tag.

Example

Letâ€™s see an example in code:

<!DOCTYPE html>
<html>
<head>
    <title>My List</title>
</head>
<body>
    <h2>My Favorite Fruits</h2>
    <ul>
        <li>Apple</li>
        <li>Banana</li>
        <li>Cherry</li>
    </ul>
    
    <h2>Top 3 Movies</h2>
    <ol>
        <li>Inception</li>
        <li>The Dark Knight</li>
        <li>Interstellar</li>
    </ol>
</body>
</html>

When you run this code, you'll see two lists: one with bullet points for fruits and one numbered list for movies.
